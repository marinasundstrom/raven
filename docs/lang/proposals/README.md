# Proposals

This directory contains proposals for language features, e.g. new syntax and behavior.

## Layout

- Active proposals stay in this directory.
- Implemented proposals move to [`implemented/`](implemented/README.md) once the documented
  surface is considered shipped enough that future changes should start from the spec and a
  fresh proposal instead of reopening the old one.
- Draft ideas live under [`drafts/`](drafts/).
- Rejected proposals live under [`rejected/`](rejected/).
