# Learn Raven

Choose the path that best matches what you already know. All of these guides
describe the same language; they differ in pace and the assumptions they make
about the reader.

## I want the shortest introduction

Start with [Raven in 60 seconds](raven-in-60-seconds.md). It presents a complete
small program and points out the language features that matter most.

Continue with the [language introduction](introduction.md) when you want a
broader tour of functions, data modeling, patterns, error handling, objects,
and .NET interoperability.

## I am new to programming

Follow [Raven for absolute beginners](raven-for-absolute-beginners.md). It
introduces values, types, decisions, loops, functions, collections, and domain
types without assuming knowledge of C# or another language.

Raven currently runs from a source checkout, so use [Install and run
Raven](getting-started.md) when you are ready to execute the examples.

## I already program in another language

Read [Raven in 60 seconds](raven-in-60-seconds.md), then continue through the
[language introduction](introduction.md). The introduction explains Raven on
its own terms without treating another language as the default.

Afterward, use the [domain-modeling guide](lang/domain-modeling.md) to choose
among functions, records, unions, classes, and interfaces in real programs.

## I use C# and .NET

Start with [Raven for C# developers](raven-for-csharp-developers.md). It maps
common C# starting points to Raven while preserving direct use of .NET types
and libraries.

Use [Install and run Raven](getting-started.md) for the current source-build
workflow and the [tool documentation](compiler/index.md) for projects, compiler
commands, diagnostics, and editor support.

## I need precise language rules

Learning guides favor motivation and examples. When you need exact syntax or
semantic rules, use the [language reference](lang/README.md). The reference
contains the normative specification chapters and the non-normative EBNF
grammar.

## Suggested order

1. [Raven in 60 seconds](raven-in-60-seconds.md)
2. [Install and run Raven](getting-started.md)
3. [Language introduction](introduction.md)
4. [Domain modeling](lang/domain-modeling.md)
5. [Style guide](lang/style-guide.md)
6. [Language reference](lang/README.md), as needed
